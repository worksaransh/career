# Career GPS AI — Product Requirements & User Journey ## Project Vision To be the "Operating System" for careers in India, providing an AI-native experience that feels like a personal career concierge. ## Core User Journey 1. **Landing**: High-impact value prop. 2. **Signup/In**: Seamless Apple/Google/LinkedIn auth. 3. **User Type Selection**: Personalized onboarding paths for Students, Professionals, Parents, etc. 4. **Onboarding**: - Language Selection. - Resume Upload (Professional path) or adaptive interest mapping. - AI Processing: Extracting data to build the "Career Twin". - Career Twin Creation: A digital mirror of the user's professional self. 5. **Dashboard**: The command center. 6. **Growth Tools**: - Weekly Goals & Reflection. - Career Timeline (past & future). - Career Upgrade Score (the north star metric). 7. **Exploration**: - Career Explorer (Roles). - College Explorer (Education). - Skill Explorer (Upskilling). 8. **Interactions**: - AI Coach Chat (Always available). - Community: Networking and mentorship. 9. **Premium**: Subscription management. ## Design Aesthetic Apple + Linear + CRED + ChatGPT. Dark mode by default with high-contrast accents. Precision-engineered data visualizations.
